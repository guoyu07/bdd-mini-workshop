# Getting started

1. Clone this project.
2. Run composer:

    ```
    composer install
    ```

3. Run behat:

    ```
    vendor/bin/behat
    ```
    
4. Fire up the build in php server to view the behat fancy html output:

    ```
    php -S localhost:8080 -t public/bdd
    ```
    
5. Try implementing the missing steps.
